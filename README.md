Sobre o Projeto
A Alyse Clinic é um projeto fictício criado do zero. 
Nome, identidade visual, logo, layout e conteúdo. 
Com o objetivo de demonstrar minha capacidade de entregar sites completos, elegantes e funcionais para clientes reais.

Este projeto simula o site de uma clínica de estética avançada, e representa exatamente o tipo de entrega que ofereço: design sofisticado, responsivo e com atenção a cada detalhe.


✨ O que foi desenvolvido
✅ Identidade visual criada do zero (nome, paleta, tipografia)
✅ Layout 100% responsivo (desktop e celular)
✅ Animações suaves com ScrollReveal
✅ Menu hamburger funcional no mobile
✅ Seções completas: Hero, Sobre, Vídeo, Procedimentos, Depoimentos, Contato e Footer
✅ Hospedagem no Netlify



Tecnologias Utilizadas



| Tecnologia | Uso |
|---|---|
| `HTML5` | Estrutura semântica das páginas |
| `CSS3` | Estilização, responsividade e animações |
| `JavaScript` | Interações, menu mobile e scroll suave |
| `ScrollReveal.js` | Animações ao rolar a página |
| `Google Fonts` | Tipografia (Cormorant Garamond, Montserrat, Raleway) |
| `Netlify` | Hospedagem e deploy contínuo |
